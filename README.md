# UrbanHeat Frontend

Este repositório contém a interface de visualização do sistema **UrbanHeat**. A aplicação é responsável por consumir a API REST do backend, processar dados em tempo real e exibir indicadores ambientais críticos.

---

## Como Executar

### 1. Pré-requisito
Certifique-se de que o backend (`UrbanHeat-Back-end`) esteja rodando na porta `8000`.
Você pode achar o link do repositório por esse link: https://github.com/Joao-Vitor-Freitas/UrbanHeat-Back-end
Lá tem instruções para configurar o ambiente

### 2. Mock dos dados
Para popular a api com dados:

1. Abra o terminal na pasta do projeto **Frontend**.
2. Execute o script:
   ```bash
   python mock.py
3. Abra o arquivo html no navegador
